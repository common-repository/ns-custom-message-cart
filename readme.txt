Add a custom message in cart page! No code required!

== Description ==
When plugin enabled you can add a message that will display in cart page.
Page option will appear in your WordPress backend menu, and message text that you can change.
You can also choose the position in cart page and a backgroud color.
Use this plugin is very quick and simple, and no code required.

Simple to use!

https://www.youtube.com/watch?v=9ynJ-FG6f4s

PREMIUM FEATURES of custom cart message:

Custom message ased on **events**
Display message in more position
Custom **style** for message
Add button to **close or dismiss** message

== Installation ==

1. Upload the entire 'ns-custom-message-cart' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'NS Custom art Message' and start to customize it.
